Introduccion.

los tipos de datos son la base de cualquier programa, ya que determinan còmo se almacenan y manipulan los valores. Existen tipos primitivos como numeros, cadenas de texto, booleanos null, undefined, simobolor y BigInt, y Estructuras de datos màs avanzados como arrays, conjuntos, mapas y objetos, Conocerlos es fundamental para escribir codigo eficiente y bien estructurado.
